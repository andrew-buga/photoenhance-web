```bash
# 1. Перейти на: https://replicate.com
# 2. Зареєструватися або увійти
```

```bash
# Dev сервер вже запущено, просто оновіть браузер
# http://localhost:3000/enhance
# Завантажте зображення і побачите різницю в якості
```

**Вартість**: ~$0.001-0.01 за upscale (оплачуєш як йдеш)
